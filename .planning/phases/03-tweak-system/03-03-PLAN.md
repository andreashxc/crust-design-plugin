<objective>
Integrate tweak rendering into the live popup. After this plan, the popup filters to the active tab scope, loads/stores tweak values, shows validation errors, and resets values to defaults.
</objective>

<tasks>

Files:

- `apps/extension/src/popup/store.ts`
- `apps/extension/src/popup/store.test.ts`

Action:

- Add slices for:
  - `tweakValues: Record<id, Record<string, unknown>>`
  - `tweakErrors: Record<id, TweakValidationError[]>`
- Add setters for both.
- Preserve existing Phase 2 slices.

Acceptance criteria:

- Store reset tests include tweak state.

Verify:

- `corepack pnpm vitest run apps/extension/src/popup/store.test.ts`

## Task 2: Filter Popup Rows By Active URL

Files:

- `apps/extension/src/popup/grouping.ts`
- `apps/extension/src/popup/grouping.test.ts`
- `apps/extension/entrypoints/popup/App.tsx`
- `apps/extension/entrypoints/popup/App.test.tsx`

- Matching regex fallback experiment is visible.

Verify:

- `corepack pnpm vitest run apps/extension/src/popup/grouping.test.ts apps/extension/entrypoints/popup/App.test.tsx`

## Task 3: Bootstrap And Sync Tweak Storage

Files:

- `apps/extension/entrypoints/popup/main.tsx`
- `apps/extension/entrypoints/popup/App.test.tsx`

Action:

- During bootstrap, load tweak values/errors for registry entries.

- Keep storage as source of truth.

Acceptance criteria:

- Storage change tests update store state.
- Bootstrap handles missing tweak values.

Verify:

- `corepack pnpm vitest run apps/extension/entrypoints/popup/App.test.tsx`

## Task 4: Wire `TweakControls` Into Experiment Row

Files:

- `apps/extension/entrypoints/popup/components/ExperimentRow.tsx`
- `apps/extension/entrypoints/popup/App.test.tsx`

Action:

- Render `TweakControls` when `entry.tweaks.length > 0`.
- On change:
  - validate merged values.
  - write valid values to storage.
  - store validation errors if invalid.
  - broadcast/reuse state-change path so content script reconciles.
- On reset:
  - clear stored values/errors.
  - reapply defaults.

Acceptance criteria:

- Tweak changes are persisted to `chrome.storage.local`.
- Invalid values show validation error and do not write malformed effective values.
- Reset clears values.

Verify:

- `corepack pnpm vitest run apps/extension/entrypoints/popup/App.test.tsx`
