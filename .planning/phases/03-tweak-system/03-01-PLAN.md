<objective>
Define the Phase 3 tweak contract at the SDK/storage/scope layer. After this plan, manifests can declare six typed tweak definitions, build validation rejects invalid declarations, local storage can persist tweak values/errors, and URL matching supports Chrome match-patterns plus regex fallback.
</objective>

<tasks>

## Task 1: Add Typed Tweak Schema To SDK

Files:

- `packages/experiment-sdk/src/index.ts`
- `packages/experiment-sdk/src/manifest.test.ts`
- `packages/experiment-sdk/src/registry.test.ts`

Action:

- Replace `tweaks: z.array(z.unknown()).default([])` with a discriminated union for:
  - `toggle`
  - `select`
  - `text`
  - `number-slider`
  - `color`
  - `multi-select`
- Export `TweakDefinition`, `TweakValueMap`, `TweakValidationError`.
- Keep `ApplyArgs.tweaks` as `Record<string, unknown>`.
- Add pure helpers:
  - `defaultTweakValues(tweaks)`
  - `validateTweakValues(tweaks, values)`
  - `mergeTweakValues(tweaks, stored)`
- Validate defaults against options/ranges/colors.

Acceptance criteria:

- All six tweak types have positive parse tests.
- Invalid defaults fail: select default not in options, multi-select default unknown option, slider default outside range, bad color, empty key.
- `RegistryEntry.tweaks` is typed as `TweakDefinition[]`.

Verify:

- `corepack pnpm vitest run packages/experiment-sdk/src/manifest.test.ts packages/experiment-sdk/src/registry.test.ts`
- `corepack pnpm typecheck`

## Task 3: Add Tweak Storage Helpers

Files:

- `apps/extension/src/shared/storage.ts`
- `apps/extension/src/shared/storage.test.ts`

Action:

- Add key constants:
  - `tweaks:<id>`
  - `tweak_error:<id>` or consolidated equivalent if simpler.
- Add helpers:
  - `getTweakValues(id): Promise<Record<string, unknown>>`
  - `setTweakValues(id, values): Promise<void>`
  - `clearTweakValues(id): Promise<void>`
  - `getTweakErrors(): Promise<Record<string, TweakValidationError[]>>`
  - `setTweakErrors(id, errors): Promise<void>`

Acceptance criteria:

Verify:

## Task 4: Add Regex Fallback Scope Matching

Files:

- `apps/extension/src/shared/url-match.ts`
- `apps/extension/src/shared/url-match.test.ts`

Action:
