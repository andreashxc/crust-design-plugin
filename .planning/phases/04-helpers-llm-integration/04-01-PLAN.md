## Task 2: Add Storage Shapes

Files:

- `apps/extension/src/shared/storage.ts`
- `apps/extension/src/shared/storage.test.ts`

Action:

- Add local settings helpers:
  - `getLlmSettings()`
  - `setProviderKey(provider, key)`
  - `clearProviderKey(provider)`
  - `setLlmDefaults(defaults)`
  - `getPublicLlmConfig()` returning no secret fields
- Add cache helpers:
  - `getLlmCache(hash)`
  - `setLlmCache(hash, value)`
  - `clearLlmCache()`
  - expired cache entries ignored.
- Add session counter helpers:
  - `getLlmSessionStats()`
  - `incrementLlmSessionStats(delta)`
  - `resetLlmSessionStats()`
- Add diagnostics helpers:
  - `getLastLlmError()`
  - `setLastLlmError(error)`
  - `clearLastLlmError()`

Acceptance criteria:

- Missing/malformed storage returns safe defaults.
- API keys are never returned by public config helper.
- Session counters live in `chrome.storage.session`; settings/cache/error live in `chrome.storage.local`.

Verify:

- `corepack pnpm vitest run apps/extension/src/shared/storage.test.ts`

## Task 3: Cache-Key Utility

Files:

- `apps/extension/src/shared/storage.ts`
- `apps/extension/src/shared/storage.test.ts`

Action:

Verify:
