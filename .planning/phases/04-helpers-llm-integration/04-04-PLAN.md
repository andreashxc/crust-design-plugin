## Task 2: Side-Effect Ledger

Files:

- `apps/extension/src/content/helpers.ts`
- `apps/extension/src/content/helpers.test.ts`

Action:

- Implement LIFO cleanup ledger.
- `injectStyle` appends style tagged with `data-exp-id` and removes on cleanup.
- `injectNode` inserts a node and removes it on cleanup if still attached.
- `waitFor` uses MutationObserver, timeout, and abort signal; disconnects on cleanup.
- `onUrlChange` registers cleanup-safe listener/patch for helper-level callbacks only.

Acceptance criteria:

- Cleanup removes injected style/node.
- Observers/listeners disconnect on cleanup and abort.
- Calling cleanup twice is safe.
